File Size and Performance

Page speed is a ranking factor, and images are often the largest assets on web pages:

  • Target size: Under 100KB for most images
  • Hero images: Maximum 300KB
  • Thumbnails: Under 20KB
  • Tools: TinyPNG, ImageOptim, or Squoosh for compression

Responsive Images

Serve appropriately sized images for different devices and screen densities:

<img src="image-400.webp" srcset="image-400.webp 400w, image-800.webp 800w, image-1200.webp 1200w" sizes="(max-width: 768px) 100vw, (max-width: 1200px) 50vw, 33vw" alt="Descriptive alt text">

Alt Text Mastery

Alt text is the cornerstone of image SEO, serving both accessibility and search engine optimization:

Writing Effective Alt Text

‚ùå Poor Examples

alt="image"

alt="IMG_12345"

alt="SEO tips marketing digital"

‚úÖ Optimized Examples

alt="Marketing team reviewing SEO analytics dashboard on laptop"

alt="Chocolate chip cookies cooling on wire rack in modern kitchen"

alt="Mountain hiker with backpack overlooking sunrise valley vista"

‚úÖ Alt Text Checklist

  • Keep it under 125 characters
  • Describe the image accurately and specifically
  • Include relevant context for your content
  • Avoid "image of" or "picture of" phrases
  • Include keywords naturally, never stuff
  • Consider the user's intent and your content goals

AI-Optimized Alt Text

With AI search engines analyzing content context, alt text should:

  • Use natural, conversational language
  • Include emotional or atmospheric context when relevant
  • Describe relationships between objects and actions
  • Support your content narrative

File Naming Best Practices

  • Use descriptive, keyword-rich names
  • Separate words with hyphens, not underscores
  • Keep names concise but descriptive
  • Include your target keyword when relevant
  • Use lowercase letters consistently

Structured Data for Images

Help search engines understand your images with structured data:

ImageObject Schema

{ "@context": "https://schema.org", "@type": "ImageObject", "contentUrl": "https://example.com/recipe-image.jpg", "description": "Fresh homemade chocolate chip cookies on cooling rack", "name": "Chocolate Chip Cookie Recipe Photo", "creator": { "@type": "Person", "name": "Chef Sarah" } }

Product Images

For e-commerce, use Product schema to enhance image search visibility:

{ "@type": "Product", "name": "Wireless Bluetooth Headphones", "image": [ "https://example.com/headphones-front.jpg", "https://example.com/headphones-side.jpg" ], "offers": { "@type": "Offer", "price": "199.99" } }

Technical Implementation

Lazy Loading

Improve page speed without sacrificing user experience:

<img src="placeholder.jpg" data-src="actual-image.jpg" loading="lazy" alt="Descriptive alt text">

Image Sitemaps

Help search engines discover your images:

<url> <loc>https://example.com/sample-page.html</loc> <image:image> <image:loc>https://example.com/image.jpg</image:loc> <image:title>Image Title</image:title> <image:caption>Image caption</image:caption> </image:image> </url>

Platform-Specific Optimization

Google Images

  • Use high-quality, original images
  • Include images in your XML sitemap
  • Ensure images are crawlable (not blocked by robots.txt)
  • Use descriptive captions near images

Social Media Optimization

Optimize for social sharing with Open Graph and Twitter Cards:

<meta property="og:image" content="https://example.com/social-image.jpg"> <meta property="og:image:width" content="1200"> <meta property="og:image:height" content="630"> <meta name="twitter:card" content="summary_large_image"> <meta name="twitter:image" content="https://example.com/social-image.jpg">
