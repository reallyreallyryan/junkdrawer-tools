Chrome Extension Permissions

Our extension requests the following permissions and here's why:

  • activeTab: Access current webpage to analyze clicked images
  • contextMenus: Add "Generate AI Alt Text" to right-click menu
  • storage: Store API keys and preferences locally
  • scripting: Detect images in complex website layouts
  • host permissions: Work on all websites where you need alt text
